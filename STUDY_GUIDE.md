# DSA Interview Preparation — Sequential Study Guide

This guide structures your preparation for technical interviews in a deliberate order.
Each topic builds on the previous one. Work through them sequentially.

Within each topic you will find:
- `CONCEPTS.md` — theory, patterns, and annotated code templates
- Solved Java examples demonstrating each core pattern
- `EXERCISES.md` — problems to attempt, ordered by difficulty, with Blind 75 problems marked

---

## How to Use This Guide

1. Read `CONCEPTS.md` fully before attempting any exercises.
2. Study the solved Java examples — understand the pattern, not just the code.
3. Open `EXERCISES.md` and attempt problems without looking at solutions.
4. If you are stuck for more than 20–30 minutes, review the concepts section for the relevant pattern, then try again.
5. After solving, review your solution for time/space complexity and simplifications.
6. Move to the next topic only after you can solve Easy problems fluently and Medium problems with minor guidance.
